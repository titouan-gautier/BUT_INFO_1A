Objectif du projet : fournir les notebook jupyter suivants

- visualisation d'indicateurs sur l'utilisation de Madoc par un étudiant
  - les jour favoris d'utilisation
	- les heures de connexion favorites
	- les cours les plus vus
	- la vue globale de ses connexions sur une plage temporelle (semestre,
				  1 mois, ...)
	- le nombre de connexion par cours
	- les actions les plus réalisées
	- les resources les plus consultées

- visualisation d'indicateurs sur l'utilisation de Madoc par les enseignants:
	- le nombre de connexion par enseignant
	- les actions les plus réalisées par enseignants
	- le nombre de connexion par cours

- visualisation d'indicateurs sur un cours d'un enseignant:
	- les resources les plus accédées
	- les jours de connexion favoris
	- les horaires de connexion favoris
	- les étudiants se connectant le plus
